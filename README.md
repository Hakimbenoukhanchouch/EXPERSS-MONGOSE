# 📦 TP CRUD Application Cloud Native (Node.js + MongoDB + EJS)
![Aperçu de l'application](.EXPRESS.PNG)
## 🎯 Présentation du projet

Ce projet est une application web CRUD (Create, Read, Update, Delete) développée avec :

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS (template engine)
- Method-Override
- Body-parser

L’objectif est de créer une application **Cloud Native simple** permettant la gestion des produits.

---

## 📚 Fonctionnalités

✔ Ajouter un produit  
✔ Afficher la liste des produits  
✔ Modifier un produit  
✔ Supprimer un produit  
✔ Interface web avec EJS
